A simple and beautiful food website made with Vanilla JavaScript. This website shows different recipes and food ideas for people who love cooking.

## About This Project
Garden of Flavor is a website about food and recipes. It helps people find new recipes and learn about cooking. The website is easy to use and looks good on all devices.

### What You Can Do Here
- Look at different recipes
- See beautiful food pictures
- Find cooking tips
- Browse food categories
- Learn about ingredients

## What I Used to Make This
- **HTML5** - For the website structure
- **CSS3** - For styling and colors
- **Vanilla JavaScript** - For interactive features (no extra libraries)

## Main Features
- **Easy to Use**: Simple navigation
- **Works Everywhere**: Looks good on phones, tablets, and computers
- **Fast Loading**: No heavy libraries
- **Clean Design**: Nice colors and layout
